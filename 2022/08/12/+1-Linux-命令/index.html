+1.Linux 命令

  • linux

常用命令

日期

  • date
  • clock
  • cal
  • uptime

文件

  • 显示当前目录和当前目录下文件

    • pwd 打印当前工作目录
    • ls -a(ll)l(ist)、ll 显示当前工作目录下的文件

  • 切换目录

    • cd 切换目录

  • 创建目录和创建文件·

    • mkdir 新建目录
    • touch 新建一个文件、查看文件修改信息

  • 删除、移动和复制

    • rm -r(emove)f(orce) 删除一个文件
    • rmdir 删除目录
    • mv 移动文件，重命名文件
    • cp 复制文件

  • 权限管理

    • chmod u+x or 775 更改文件权限 ugo权限
    • chown 更改文件所属者
    • chgrp 更改文件所属组
    • srwx
    • umask
    • df

  • 显示信息

    • shell 编写shell脚本

    • vim 文件编辑器

    • cat 显示文本文件

    • nl 显示文本文件

    • more

    • less

    • head 从前往后显示

    • tail -n 从后往前显示，显示行数

  • 查找

    • which 查看文件路径
    • whereis
    • locate
    • find -time… 、-name 查找包

  • 压缩

    • zip
    • unzip
    • gzip
    • tar -c or z(ip)xvf 解压

系统设备

  • 查看系统信息

    • sh
    • cat /etc/redhat-release
    • unam -a 查看系统内核版本
    • df -hT、-lh 查看磁盘空间情况
    • du 查看当前占用空间
    • ps aux、-ef是否有进程、-ef|grep筛选相应进程
    • kill 杀死进程
    • top 查看相应服务是否启动
    • free -m 查看内存情况
    • echo 1 or 2 or 3 > /proc/sys/vm/drop_caches 清除缓存
    • vmstat
    • iostat
    • lsof

  • 开关机

    • shutdown -rh now 立即关机、time 准确时间[20:00],限时[10 min]
    • reboot 重启
    • poweroff 关机
    • init 0 or 6 系统的启动级别

网络

    • ip -a 查看网卡信息
    • ifconfig -a 网络设备信息
    • ifup 启用网卡
    • ifdown 关闭网卡
    • netstat -tupln、-ntlp查看占用端口
    • cul 访问相应端口
    • telstat
    • dig
    • route
    • trace
    • ping 测试网络连接
    • nmcli con sh or connection reload 网络管理工具，重启网卡
    • nmtui nmcli网络管理工具的图形化界面工具
    • ssh 远程连接服务器
    • git 从GitHub拉取项目代码
    • wget 获取网络地址上的文件压缩包
    • sz 下载文件到本地

其他

    • ln 与目标文件产生链接
    • diff 比较两个文件的不同
    • grep 筛选字符串
    • wc
    • watch

  • 计划任务

    • at 在某一时刻执行设置好的命令
    • crontab 在某些时间段执行设置好的命令 时间可以用确切的，也可以用虚指的，比如morning
    • nohup &后台启动
    • jobs 任务查询

  • 包管理工具(安装软件)

    • yum -y这样就不用输入y了
    • dnf yum这种包管理工具的升级版
    • apt(ubuntu使用)
    • –version 查看版本

  • 帮助

    • –help 获取某个命令的帮助
    • man 一个命令手册可以查到详细的命令使用方式和作用
    • info 查看详细的命令
    • yum provides 查看哪些软件提供这种服务

  • 操作

    • history -c、Ctrl+r 查看历史命令
    • ！！
    • ！command
    • ！num
    • ！？command
    • [1..3]

用户

  • 查看用户
    • id 显示用户id
    • who 显示用户账户
    • w who的缩写
  • 切换用户，切换用户做什么
    • shadow
    • su - （username）切换某个账户，低权限账户转高权限账户需要输入密码，反之不需要
    • sudo 获得root权限执行命令
  • 增删改，设置和修改密码
    • useradd 创建一个新的用户
    • userdel -r 删除用户 ，没有参数r容易删不干净
    • usermod 修改用户信息
    • groupadd 在一个组中新增新用户
    • passwd 设置密码或者修改某个用户的密码
  • 睡眠注销
    • sleep 睡眠
    • logout 注销
    • exit 注销，推出账户
